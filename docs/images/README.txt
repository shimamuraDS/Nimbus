Put the following images in this directory (docs/images/):

1. logo.png
   - App icon for README header (96x96 or larger, PNG with transparency)
   - Source: Nimbus.ico or any app icon design
   - Used at top of README

2. screenshot-main.png
   - Main app screenshot showing the full window with Today weather view
   - Recommended: 1280x720 or similar 16:9 ratio
   - Shows: hourly timeline, toolbar, navigation buttons, the "今日天气" title

3. screenshot-today.png
   - Today view: hourly weather timeline with current hour highlighted
   - Shows: horizontal scrolling hourly cards with current hour cyan glow

4. screenshot-past.png
   - Past 7 days view: warm coral-colored weather cards
   - Shows: 7-day history with navigation button

5. screenshot-settings.png
   - Settings view: auto-start toggle, API settings panel, alert time list
   - Shows: the full settings UI with collapsible API section

6. icon-standard.png
   - Small badge icon for Standard version (24x24, PNG)
   - E.g. a simplified weather icon or "S" badge

7. icon-ai.png
   - Small badge icon for AI version (24x24, PNG)
   - E.g. a simplified brain/AI icon or "AI" badge
