Put the following images in this directory (docs/images/):

1. logo.png
   - App icon for README header (96x96 or larger, PNG with transparency)
   - Used at top of README

2. screenshot-today.png
   - Today view: hourly weather timeline with current hour highlighted
   - Shows: horizontal scrolling hourly cards, "今日天气" title (purple)

3. screenshot-past.png
   - Past 7 days view: warm coral-colored weather cards
   - Shows: 7-day history with "过去 7 天" title (coral)

4. screenshot-future.png
   - Future 7 days view: cyan-colored forecast cards
   - Shows: 7-day forecast with "未来 7 天" title (cyan)

5. screenshot-alerm.png
   - Settings alert view: auto-start toggle, alert time list with add/modify/delete
   - Shows: Flickable settings page with time picker and alert configuration

6. screenshot-standard.png
   - Standard version screenshot showing key feature (e.g. settings or about)
   - Used in feature preview row

7. screenshot-ai.png
   - AI version screenshot showing LLM settings or AI-generated notification
   - Used in feature preview row

All screenshots recommended at ~400x300 or similar aspect ratio.
