# 用户使用手册

## 1. 安装与启动

1.  双击运行 `WeatherApp_Setup.exe` 安装包。
2.  安装完毕后，程序将默认开启**开机自启动**并隐藏运行。
3.  您将在 Windows 屏幕右下角（系统托盘区域）看到一枚天气助手的图标。
4.  点击该图标，即可在屏幕右下角唤出天气提醒控制面板。

## 2. 各页面操作方法

### 2.1 查看今日逐小时天气
唤出面板后，首页以横向时间线展示未来 24 小时的逐小时天气和气温。当前小时的卡片以蓝色高亮并略微放大。若卡片超出屏幕范围，可左右滑动浏览。

### 2.2 查看历史天气
在首页点击左侧的 ◀ 按键，页面从左往右滑入，展示过去 7 天每天的早晚气象记录（暖珊瑚色卡片背景）。默认显示最近一天的记录，可左右滑动浏览全部 7 天。点击右侧的 ▶ 即可返回首页。

### 2.3 查看未来预报
在首页点击右侧的 ▶ 按键，页面从右往左滑入，展示未来 7 天每天的早晚预测数据（冷青色卡片背景）。可左右滑动浏览全部 7 天。点击左侧的 ◀ 即可返回首页。

## 3. 系统设置与预警提醒设置

在界面的顶部工具栏右侧，点击 **设置** 按键可进入配置页面（页面从顶部向下滑入，返回时从下向上滑出）。设置页面内容较多时可上下滚动浏览。

### 3.1 开机自启动设置

页面顶部设有"开机自启动"开关（电光青色滑动开关）：
- **启用**（默认）：应用将在 Windows 启动时自动静默运行到系统托盘。
- **禁用**：关闭开关即可取消开机自启动，注册表项将被移除。

### 3.2 更正定位

**自动定位模式**（默认）：
- 界面显示 "自动定位：【当前城市】" 和可点击的 "定位不准？" 链接。
- 点击 "定位不准？" 切换到手动模式。

**手动定位模式**：
- 界面显示 "选择城市：" 标签和城市下拉选择框（覆盖全国 98 个主要城市）。
- 从下拉框中选择您的城市，天气数据将立即更新。
- 点击 "返回自动定位" 可恢复 IP 自动定位。

### 3.3 配置天气定时提醒

1.  在页面底部的小时/分钟下拉选择器中，分别选择小时（00~23）和分钟（00~59）。
2.  在"提前"选择器中，设置提前提醒的时间量（0~23小时 + 0~59分钟）。例如设置为"30分钟"，则系统会在设定提醒时间前30分钟发出通知，告知该时间点之后的天气情况。
3.  点击 **添加** 按钮完成添加。每个提醒时间旁边会显示对应的提前量（如"提前30分钟"或"提前1小时"）。
4.  如需修改某时段，点击该时间点旁边的 **修改**，调整时间和提前量后点击 **保存**。
5.  如需取消某时段的提醒，点击该时间点旁边的 **删除** 即可。

### 3.4 返回主页
操作完毕后，点击工具栏右上方的 **返回** 即可保存设置并回到主页。

## 4. 托盘操作

*   **左键单击托盘图标**：显示/隐藏天气面板。
*   **右键单击托盘图标**：弹出菜单，可选择「显示窗口」或「退出」程序。

## 5. 常见问题

**Q: 天气数据不更新？**
A: 请检查网络连接，并确认 `config.ini` 中的 API 密钥是否有效。

**Q: 自动定位不准确？**
A: IP 定位精度有限，请在设置页面点击"定位不准？"手动选择正确的城市。

**Q: 没有收到天气预警通知？**
A: 请确认已在设置中添加了提醒时间点，且当前时间段天气非晴天（晴天不触发提醒）或存在官方预警。
