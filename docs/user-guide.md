# 用户使用手册

## 1. 安装与启动

1.  双击运行 `WeatherApp_Setup.exe` 安装包。
2.  安装完毕后，程序将默认开启**开机自启动**并隐藏运行。
3.  您将在 Windows 屏幕右下角（系统托盘区域）看到一枚天气助手的图标。
4.  点击该图标，即可在屏幕右下角唤出天气提醒控制面板。

## 2. 各页面操作方法

### 2.1 查看今日天气
唤出面板后，首页中央的卡片即展示本地今日白天与夜晚的详细气象数据（天气、气温、湿度）。

### 2.2 查看历史天气
在首页点击左侧的 `<` 按键，列表将展示过去 7 天每天的早晚气象记录。点击右侧的 `>` 即可返回首页。

### 2.3 查看未来预报
在首页点击右侧的 `>` 按键，列表将展示未来 7 天每天的早晚预测数据。点击左侧的 `<` 即可返回首页。

## 3. 系统设置与预警提醒设置

在界面的顶部工具栏右侧，点击 **设置** 按键可进入配置页面：

### 3.1 更正定位
若您发现天气城市与当前所在不符，可点击蓝字 `定位不准？`，系统将弹出手动的城市选择列表供您指定。选定城市后，也可点击"恢复自动定位"切换回 IP 自动定位。

### 3.2 配置恶劣天气定时提醒

1.  点击页面底部的 **【添加时间点】** 按钮。
2.  在弹出的时间选择器中，设置您习惯接收预警的时间（例如出门前的 07:30）。
3.  添加成功后该时间点将出现在上方列表中。若某日该时段程序检测到极端恶劣天气（如暴雨、强沙尘暴等）或官方预警下发，屏幕右下角将主动弹出防灾警告。
4.  如需取消某时段的提醒，点击该时间点旁边的 **【删除时间点】** 即可。

### 3.3 返回主页
操作完毕后，点击工具栏右上方的 **返回** 即可保存设置并回到主页。

## 4. 托盘操作

*   **左键单击托盘图标**：显示/隐藏天气面板。
*   **右键单击托盘图标**：弹出菜单，可选择「显示窗口」或「退出」程序。

## 5. 常见问题

**Q: 天气数据不更新？**
A: 请检查网络连接，并确认 `config.ini` 中的 API 密钥是否有效。

**Q: 自动定位不准确？**
A: IP 定位精度有限，请在设置页面手动选择正确的城市。

**Q: 没有收到天气预警通知？**
A: 请确认已在设置中添加了提醒时间点，且当前时间段确实存在恶劣天气或官方预警。
